# nu:speak

AI Agent 기반 맞춤형 뉴스 서비스

## 주요 기능
- 개인화 뉴스 피드
- AI 요약 카드
- 사용자 인증 및 온보딩

## 기술 스택
- Frontend: React, Next.js, Tailwind CSS
- Backend: Node.js, Express
- AI: OpenAI API
- DB: PostgreSQL, MongoDB (MVP에서는 Mock 데이터)

## 실행 방법

### 백엔드
```
cd backend
npm install
npm start
```

### 프론트엔드
```
cd frontend
npm install
npm run dev
```
